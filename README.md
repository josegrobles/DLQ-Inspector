# How to run

### First install the packages with:

`npm install`

### Once all the packages are installed run it:

You will need the following Environment variables (Local):

```
LIBER_ENV
CLIENT_ID (Swell)
CLIENT_KEY (Swell)

TOPIC_PROFILE
TOPIC_KEY

JWT_SECRET
JWT_EXPIRATION
```

Now run:

`npm start`

### To get the API Docs go to:

`GET /api-docs`
